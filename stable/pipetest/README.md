# bpkg

Bpkg is the BASH package manager

## Installation

```bash
$ neo install bpkg
```

```bash
$ curl -sL https://git.io/get-bpkg | bash -
```

## Url shortening

```bash
curl -i "https://git.io" \
     -d "url=https://raw.githubusercontent.com/zionrc/hub/master/stable/pipetest/install.sh" \
     -d "code=get-pipetest"
```
