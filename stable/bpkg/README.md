# bpkg

Bpkg is the BASH package manager

## Installation

```bash
$ neo install bpkg
```

```bash
$ curl -sL https://git.io/get-bpkg | bash -
```

## Url shortening

```bash
curl -i https://git.io \
     -F "url=https://raw.githubusercontent.com/zionrc/hub/master/stable/bpkg/install.sh" \
     -F "code=get-bpkg"
```
